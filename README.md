# Usage

1. Clone this repo

`git clone https://github.com/donchriscorleone/assessment-app.git`

2. Go to the root directory

`cd assessment-app`

3. Install the dependencies

`npm install`
or
`yarn install`

4. Run the development server

`npm run dev`
